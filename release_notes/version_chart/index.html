Version chart

This version chart shows which versions of our standard Prophecy Scala Libraries and Prophecy Python Libraries should be used with any given version of the Prophecy product. In general Prophecy strives to be backwards compatible, so older sets of Prophecy Libs should be able to run on a newer version of Prophecy.

caution

Check to make sure the Spark version and Scala version of your Prophecy Scala Libs match the Spark and Scala versions of your cluster!

You can query this table using the following API and your personal access token:

curl  --header 'X-Auth-Token: $PROPHECY_PAT' --location https://app.prophecy.io/api/editor/plibVersions
note

Prophecy versions that are labeled with EM are Extended Maintenance releases. For more information, see Prophecy versions support.
